INFO Commands

Use [@Panais#5582 help] for full commands list.

Commands Description
avatar Be able to view the avatar of a server member.
help Returns all Commmands, or one specific command.

MUSIC Commands

Commands Description
play Plays a Song/Playlist in your VoiceChannel.
playskip Plays a Song/Playlist and skips.
playtop Plays a Song/Playlist and adds it to the Top.
skip Skips the Current Track
mix Plays a defined Mix
autoplay Toggles Autoplay
addrelated Add a similar/related song to the current Song.
stop Stops playing and leaves the Channel.

FILTER Commands

Commands Description
addfilter Add a Filter to the Filters.
clearfilters Clears all Filters
custombassboost Sets a custom Bassboost with Gain.
customspeed Changes the Speed of the Song.
filters List all active and possible Filters.
removefilter Removes a Filter from the Queue.
setfilter Sets (Overwrides) all Filters.

QUEUE Commands

Commands Description
clearqueue Clears the Queue
jump Jumps to a specific Song in the Queue.
list Lists the current Queue.
loop Enable/Disable the Song- / Queue-Loop.
move Moves one Song to another Place.
previous Plays the previous Song
remove Removes one+ Song(s)
shuffle Shuffles (Mixes) The Queue.
status Shows the Queue Status
unshuffle Un-Shuffles (Mixes) The Queue.
volume Adjusts the Volume of the Music.

SETTINGS Commands

Commands Description
autoresume Enable/Disable the Autoresume for this Guild, if the bot restarts.
botchat Shows Information about the Bot.
defaultautoplay Defines if Autoplay should be enabled on default or not.
defaultfilter Defines the Default Filter(s)
defaultvolume Defines the Default Volume of the Bot.
dj Manages the Djs
prefix Changes the Prefix of the Bot.

SONG Commands

Commands Description
addend Adds this Song back to the end of the Queue.
forward Forwards for X Seconds
grab Jumps to a specific Position in the Song.
lyrics Sends the Song Lyrics.
nowplaying Shows the current Playing Song.
replay Replays the current song
rewind Rewinds for X Seconds
seek Jumps to a specific Position in the Song.
